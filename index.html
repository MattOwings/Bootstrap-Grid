Goal: Make everything easy to read no matter what your screen size is
Extra Large Display: "Important" has a much larger display than "Less"
Larger screen size lets you have "important" elements larger while still being able to see "less" important ones
Large Display: "Important" has a larger display than "Less"
Screen size is still large enough to display "important" elements larger than "less" important ones
Medium Display: Become only one "Important" and one "Less" per row. "Important" is larger than "Less"
Screen is small enough to reduce the number of elements in each row to improve readability
Small / Mobile Device Display: Becomes a column, one cell per row
Mobile device screens (or very small computer screens) require the elements to become columns to vastly improve readability
